## Uber Eats EDA Project

This project performs exploratory data analysis on a synthetic food delivery dataset inspired by Uber Eats.

### Objectives
- Analyze delivery performance
- Identify rating drivers
- Study surge pricing behavior

### Tools
- Python
- pandas
- matplotlib / seaborn
- uv

### Key Findings
- Peak-hour orders experience longer delivery times
- Traffic level significantly impacts delays
- Higher prep times reduce customer satisfaction
